# Windows 实时字幕捕获工具

## 功能概述

本工具能够捕获并提取 Windows 操作系统自带的实时字幕功能所显示的文字内容。

### 主要功能

- ✅ **实时监测**：自动检测并获取 Windows 实时字幕窗口中的文本信息
- ✅ **自动保存**：将捕获到的字幕内容自动保存到本地文件系统
- ✅ **历史记录**：按日期查看和管理已保存的字幕历史
- ✅ **复制功能**：一键复制当前字幕内容或全部历史记录
- ✅ **导出功能**：将历史记录导出为文本文件
- ✅ **稳定运行**：低资源占用，不影响系统性能

## 技术原理

本工具通过 `pywinauto` 库调用 Windows UI Automation API 实现对实时字幕窗口的访问：

1. 使用 UIA 模式连接标题为"实时辅助字幕"或"Live Captions"的窗口
2. 通过控件 ID (`CaptionsTextBlock`) 定位字幕文本元素
3. 提取控件中的文本内容并进行实时更新

## 系统要求

- **操作系统**：Windows 10 1903 及以上版本 / Windows 11
- **Python 版本**：Python 3.7 及以上
- **依赖库**：
  - `pywinauto`：用于 UIA 自动化操作
  - `pyperclip`：用于剪贴板操作

## 安装步骤

### 方法一：直接运行（推荐）

1. 确保已安装 Python 3.7+
2. 安装依赖库：
   ```bash
   pip install pywinauto pyperclip
   ```
3. 双击运行 `启动字幕捕获工具.bat`

### 方法二：手动运行

```bash
cd SubtitleCapture
pip install pywinauto pyperclip
python subtitle_capture.py
```

## 使用说明

### 准备工作

1. 确保 Windows 实时字幕功能已开启
2. 使用快捷键 `Win + Ctrl + C` 可以快速开启/关闭实时字幕

### 操作指南

1. **启动捕获**：点击"开始捕获"按钮，工具将自动监测字幕窗口
2. **停止捕获**：点击"停止捕获"按钮，停止监测
3. **查看当前字幕**：在"当前字幕内容"区域查看实时捕获的文本
4. **复制当前内容**：点击"复制当前内容"按钮复制当前字幕
5. **查看历史**：从日期下拉框选择日期，查看该日期的历史记录
6. **复制全部历史**：点击"复制全部历史"按钮复制所有历史记录
7. **导出历史**：点击"导出历史记录"按钮将历史记录保存为文件
8. **清除历史**：点击"清除历史记录"按钮删除选中日期的记录
9. **打开存储文件夹**：点击"打开存储文件夹"查看保存的文件

### 存储位置

字幕历史记录保存在以下路径：
```
Documents\SubtitleCapture\subtitle_YYYYMMDD.txt
```

## 注意事项

1. 首次使用时需要先开启 Windows 实时字幕功能
2. 工具仅能捕获 Windows 官方实时字幕窗口的内容
3. 建议以普通用户权限运行，无需管理员权限
4. 工具运行时资源占用极低，不会影响系统性能
